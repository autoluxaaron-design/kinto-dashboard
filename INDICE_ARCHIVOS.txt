📄 ARCHIVOS PRINCIPALES (APLICACIÓN)
────────────────────────────────────

1. 📌 index.html [RECOMENDADO - ABRE ESTO]
   ├─ Tipo: Aplicación web HTML/CSS/JavaScript
   ├─ Descripción: Dashboard interactivo completo, funciona en cualquier navegador
   ├─ Características:
   │  ├─ Carga datos desde Google Sheets automáticamente
   │  ├─ Filtros dinámicos (Lugar, Vehículo, Estado, Búsqueda)
   │  ├─ Códigos de color por ubicación
   │  ├─ Alertas visuales por estado de próxima reserva
   │  ├─ Barra de estadísticas en tiempo real
   │  ├─ Diseño responsivo (desktop, tablet, móvil)
   │  └─ 100% funcional sin servidor
   ├─ Cómo usar:
   │  ├─ 1. Haz doble clic en el archivo
   │  ├─ 2. Se abrirá automáticamente en tu navegador
   │  └─ 3. Los datos se cargarán en 2-3 segundos
   ├─ Requisitos: Navegador web (Chrome, Firefox, Edge, Safari)
   └─ Nota: Este es el formato recomendado por su facilidad

2. 📌 app.py [ALTERNATIVA - PYTHON/STREAMLIT]
   ├─ Tipo: Aplicación Python con Streamlit
   ├─ Descripción: Alternativa si tienes problemas con CORS en la versión HTML
   ├─ Características:
   │  ├─ Interfaz moderna con Streamlit
   │  ├─ Mismo funcionalidad que index.html
   │  ├─ Versión en tarjetas disponible
   │  ├─ Exportación a CSV y Excel
   │  ├─ Cache automático de datos
   │  └─ Mejor manejo de CORS
   ├─ Cómo usar:
   │  ├─ 1. Instala dependencias: pip install streamlit pandas
   │  ├─ 2. Ejecuta: streamlit run app.py
   │  ├─ 3. Abre http://localhost:8501 en tu navegador
   │  └─ 4. ¡Listo!
   └─ Requisitos: Python 3.7+ instalado

📞 PREGUNTAS FRECUENTES
──────────────────────

P: ¿Cómo abrirlo?
R: Haz doble clic en index.html (se abrirá en tu navegador predeterminado)

P: ¿Necesito servidor?
R: No, funciona localmente en tu navegador

P: ¿Cómo actualizo los datos?
R: Recarga la página (Ctrl+R). Automáticamente obtiene los últimos datos

P: ¿Qué navegadores soporta?
R: Chrome, Firefox, Edge, Safari (cualquier navegador moderno)

P: ¿Es responsivo (funciona en móvil)?
R: Sí, se adapta automáticamente a cualquier tamaño de pantalla

P: ¿Debo instalar algo?
R: No, excepto si usas la versión Python (requiere Python 3.7+)

P: ¿Puedo modificar los colores?
R: Sí, ve a CONFIG_AVANZADA.txt y busca la sección "PERSONALIZACIÓN DE COLORES"

P: ¿Qué pasa si el Google Sheet cambia?
R: Automáticamente se refleja en la siguiente recarga de la página
